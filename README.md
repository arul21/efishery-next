### Usage

```javascript
git clone https://github.com/arul21/efishery-next.git
```

```javascript
yarn install
or
npm install
```

Runs the app in the development mode

```javascript
yarn dev
or
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view it in the browser.
