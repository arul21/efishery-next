# Commodity Dashborad

Preview https://efishery-next.vercel.app/

### Usage

```bash
git clone https://github.com/arul21/efishery-next.git
```

```bash
yarn install
# or
npm install
```

Runs the app in the development mode

```bash
yarn dev
# or
npm run dev
```

Open [localhost](http://localhost:3000) to view it in the browser.
